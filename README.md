## Fastest APIs 
Fastest Way to Create APIs using NestJS Framework and MYSQL Server.

 ## Description
Simple CRUD API for users in minute without any handling of endpoints or db collection

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
